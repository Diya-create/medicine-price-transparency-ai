# Requirements Document

## Problem
Patients overpay for medicines due to lack of generic awareness and prescription-level price transparency.

## Objective
Build an AI system that:
- Extracts medicines from prescription
- Maps branded medicines to generic equivalents
- Shows price comparison and savings
- Locates nearest Jan Aushadhi store

## Functional Requirements
- Prescription upload (image/PDF)
- OCR-based medicine extraction
- Salt identification
- Brand vs Generic comparison
- Savings calculation
- Store locator integration

## Non-Functional Requirements
- Secure data handling
- Scalable cloud deployment
- Fast processing time
