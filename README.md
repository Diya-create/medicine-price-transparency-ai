# Medicine Price Transparency AI

## Problem
Patients overpay for medicines due to lack of generic awareness and prescription-level price transparency.

## Solution
An AI-powered system that:
- Extracts medicines from prescription (OCR)
- Maps branded medicines to generic equivalents
- Compares prices and calculates savings
- Stores price data transparently using Starknet (Cairo smart contract)

## Tech Stack
- Python (OCR + AI processing)
- Medicine database mapping
- Cairo smart contract (Starknet)
- GitHub for version control

## Future Scope
- Real-time pharmacy integration
- Insurance comparison
- National price transparency dashboard
